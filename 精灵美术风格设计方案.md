## 三、统一风格规范

### Palworld 风格锚点
- 圆润关节无棱角
- 眼睛占脸比 ≥35%
- 动态重心偏前（显活泼）
- 所有阴影带轻微扩散柔边

### 渲染关键词
- `Palworld style`
- `low-poly`
- `cute chibi`
- `3D game asset`
- `octane render`
- `soft lighting`
- `front-facing full-body view`
- `simple solid background`
- `newborn chibi scale`

### 建模规范（后续开发用）
- 顶点数 ≤800，面数 ≤600
- 所有角色/道具均含3级LOD
- 关键特征线保留手绘感硬边
- matte clay texture
- hand-drawn line overlay
- soft volumetric rim light
- low-poly clean topology
