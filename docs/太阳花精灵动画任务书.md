# 太阳花精灵动画任务书

> 角色：向日葵（sunflower） · 系别：植物系（plant）
> 动画类型：待机动画（idle）+ 升级动画（upgrade）
> 生成管线：wanx2.1-i2v-turbo 图生视频 → 12fps 抽帧 → rembg 抠图 → 透明 PNG 帧序列

## 二、升级动画（upgrade）任务

> 升级动画为**非循环**动画。首帧为当前等级的最终形态，尾帧为下一等级的初始形态。
> 中间过程包含升级光效、形态变化（花瓣增多、体型增大等）。

## 五、技术注意事项

1. **循环动画首尾衔接**：idle 类动画的首帧和尾帧在视觉上必须完全一致，确保循环时无跳帧感
2. **升级动画去重阈值更低**（0.90 vs 0.95）：升级过程形态变化大，允许更多帧被保留
3. **尺寸切换点**：L4→L5 升级动画需要从 256 过渡到 512，生成时注意目标输出尺寸为 512×512
4. **抠图检查**：每批帧文件生成后，用 rembg 抠图前检查视频边缘是否干净，有残留需重新生成
5. **帧文件命名**：`frame_000.png`、`frame_001.png`... 三位数补零，序号从 0 开始
6. **每目录至少 2 帧**：AI 生成的视频经过去重后可能帧数不足，需确保最终输出至少 2 帧，推荐 12-24 帧
7. **透明背景**：所有帧输出必须透明背景，不能有白色或彩色底色
