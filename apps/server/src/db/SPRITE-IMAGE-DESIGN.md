# 精灵形象图层叠加设计方案

## 核心设计原则

### 0级 = 能量球形态
- 3系通用形状（半径24的圆形光球 + 内部白色光点 + 系别元素）
- **植物系**：绿色光球 + 小叶子元素
- **动物系**：橙色光球 + 小尾巴元素
- **元素系**：浅蓝光球 + 气流线条

### 0→1 = 破壳而出
- 能量球碎裂 → 基础形态精灵出现

### 1→8 = 逐级丰富（图层叠加）
每级 = 前一级所有图层 + 1个新增图层

| 等级 | 新增图层 | 植物系示例 | 动物系示例 | 元素系示例 |
|------|---------|-----------|-----------|-----------|
| Lv.1 | 基础身体 | 茎干+花盘+眼睛 | 圆身体+三角耳+眼睛 | 半透明圆+表情 |
| Lv.2 | 小配件 | 5片小花瓣 | 白色耳尖 | 小气流环绕 |
| Lv.3 | 特征强化 | 头顶嫩芽 | 大尾巴 | 顶部旋风 |
| Lv.4 | 肢体强化 | 叶子手臂 | 小爪子 | 气流小手 |
| Lv.5 | 发光层 | 黄色光晕 | 橙色光晕 | 蓝色光晕 |
| Lv.6 | 粒子层 | 金色粒子漂浮 | 橙色粒子漂浮 | 蓝色粒子漂浮 |
| Lv.7 | 宝石层 | 额头绿色菱形 | 额头橙色菱形 | 额头蓝色菱形 |
| Lv.8 | 翅膀层 | 叶子翅膀 | 能量翅膀 | 气流翅膀 |

### 8→9 = 满级特殊
- Lv.8 + 头顶旋转光环（满级专属）
- 植物系：金色光环
- 动物系：金色光环
- 元素系：蓝色光环

## 图层叠加结构

```
Lv.9 精灵完整形象
├── 发光层（最底层，r=32 和 r=28 半透明圆）    ← Lv.5 触发
├── 翅膀层（背后）                               ← Lv.8 触发
├── 基础身体层（茎干/身体+花盘/脸+眼睛+嘴巴）     ← Lv.1 基础
├── 配件层（花瓣/耳尖/气流）                     ← Lv.2 触发
├── 特征层（嫩芽/尾巴/旋风）                     ← Lv.3 触发
├── 肢体层（手臂/爪子/小手）                     ← Lv.4 触发
├── 粒子层（周围漂浮小点）                       ← Lv.6 触发
├── 宝石层（额头菱形）                           ← Lv.7 触发
└── 光环层（头顶，最上层）                       ← Lv.9 触发
```

## 成本节省

| 项目 | 原方案 | 叠加方案 | 节省 |
|------|--------|---------|------|
| 每系图片数 | 10张（0-9级各1张） | 10张（0级+1-9级叠加） | 相同数量 |
| 独特设计 | 30张完全不同的设计 | 3系 × 1基础 + 8个通用叠加逻辑 | 设计可复用 |
| AI提示词 | 30个独立提示词 | 10个提示词模板 × 3系调色 | ~60% |

> 虽然SVG占位图数量相同（每级1张），但实际AI生图时，可以利用图层叠加逻辑，只需生成基础层和叠加层，前端通过CSS组合。

## 升级动画设计

| 升级 | 动画效果 |
|------|---------|
| 0→1 | 破壳：能量球从顶部裂开，光芒射出，新精灵出现做伸展动作 |
| 1→2 | 闪光 + 体型微膨胀 + 新配件出现 |
| 2→3 | Lv.1→2 + 小星星出现 + 特征强化特效 |
| 3→4 | Lv.2→3 + 旋转一圈 + 新肢体出现动画 |
| 4→5 | Lv.3→4 + 光晕扩散（发光层首次出现） |
| 5→6 | Lv.4→5 + 粒子从身体爆发 |
| 6→7 | Lv.5→6 + 宝石闪光 + 额头出现特效 |
| 7→8 | Lv.6→7 + 翅膀展开动画 |
| 8→9 | **满级特殊**：飞向空中 + 光环出现 + 全屏金色光芒 + 精灵落地庆祝 |
