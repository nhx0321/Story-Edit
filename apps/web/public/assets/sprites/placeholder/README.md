# 占位资产

本目录包含开发阶段使用的占位文件。所有占位资产均为 SVG 格式，带有「占位图」标记。

## 用途

- 在真实美术资产制作完成前，让前端精灵组件可以正常显示
- 数据库 `art_assets` 表注册所有 assetKey 的占位记录
- 管理员后台显示「占位」标签，提示后续替换

## 占位文件列表

占位文件已直接放置在对应目录中（如 `characters/plant/sunflower/L0.svg`），不在本目录单独存放。

## 替换流程

1. 制作真实 PNG/SVG/Lottie 资产
2. 放置到对应的目录路径
3. 在 `/admin/art-assets` 后台注册或更新资产
4. 发布资产后前端即时生效
