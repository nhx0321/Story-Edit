# 桌面精灵美术资产

本目录存放 Story Edit 平台桌面精灵系统的全部美术资产。

## 目录结构

```
assets/sprites/
├── characters/           # 角色形象
│   ├── {系别}/
│   │   └── {种类}/
│   │       ├── L0.png ~ L9.png    # 0-9级静态图
│   │       ├── egg.png            # 精灵蛋
│   │       └── animations/        # 动画资源（Lottie JSON）
│   │           ├── idle/          # 日常待机
│   │           ├── upgrade/       # 升级
│   │           ├── interact/      # 交互
│   │           └── special/       # 特殊
│   └── ...
├── items/                # 道具
│   ├── icons/            # 道具图标（64x64）
│   ├── animations/       # 道具动画
│   │   ├── use/          # 使用动画（可复用）
│   │   └── obtain/       # 获得动画（可复用）
│   └── shop/             # 商城专属资产
├── effects/              # 通用特效
│   ├── particles/        # 粒子效果（Lottie JSON，全局复用）
│   └── overlays/         # 光效叠加（SVG/PNG）
├── ui/                   # UI 元素
│   ├── badges/           # 等级徽章
│   └── icons/            # 系统图标
└── placeholder/          # 占位资产
```

## 当前系别与种类

| 系别 | 种类编码 | 名称 | 颜色主题 |
|------|---------|------|---------|
| plant | sunflower | 向日葵 | 绿色 (#4CAF50) |
| animal | fox | 小狐狸 | 橙色 (#FF9800) |
| element | wind | 小风灵 | 蓝色 (#03A9F4) |

> 系别和种类均可后续扩展。新增系别只需在 `characters/` 下新建目录并注册资产。

## 资产复用策略

### 角色形象 — 不可复用（每个精灵独立）

| 资产类型 | 复用性 | 说明 |
|---------|--------|------|
| 静态形象 (L0-L9) | **独立** | 每个精灵每个等级需独立绘制。升级是在上一级基础上增加细节和图层，但最终输出为独立 PNG |
| 精灵蛋 (egg) | **独立** | 每个精灵有自己的蛋形象 |
| 升级动画 | **独立** | 每个精灵的升级动画不同（破壳、成长等），需单独制作 |
| 待机动画 (idle) | **独立** | 不同精灵体型和动作不同，需独立动画 |
| 交互动画 | **独立** | 点击/拖拽等动作因精灵形象不同而不同 |

**等级递增设计原则**：L(n+1) 在 Ln 的基础上增加细节、体型放大、色彩更丰富。但输出为独立静态 PNG，不做图层叠加（前端不解析图层）。

### 道具动画 — 可复用

| 资产类型 | 复用性 | 说明 |
|---------|--------|------|
| 道具使用动画 (use) | **全局复用** | 如浇水、修剪等动作动画，所有精灵共用同一套使用动画。动画表现为道具本身的动态效果（如洒水壶喷出水流）|
| 道具获得动画 (obtain) | **全局复用** | 所有道具获得时的弹窗/飞入动画共用 |
| 道具图标 | **独立** | 每个道具有独立图标 |

> 如需为特定精灵定制道具使用效果（如向日葵浇水 vs 狐狸浇水效果不同），则为该精灵创建专属动画，assetKey 设为 `item/{code}/use-{species}`。

### 通用特效 — 全局复用

| 资产类型 | 复用性 | 说明 |
|---------|--------|------|
| 粒子效果 | **全局复用** | 星星爆发、闪光等粒子效果全局共用 |
| 光效叠加 | **全局复用** | 升级光效、孵化光芒等全局共用 |

### 复用决策树

```
需要新资产？
  ├── 是角色形象？ → 独立（每种精灵 × 每个等级）
  ├── 是升级动画？ → 独立（每种精灵 × 每次升级）
  ├── 是待机动画？ → 独立（每种精灵）
  ├── 是道具使用/获得动画？ → 复用（全局共用一套）
  ├── 是粒子/光效？ → 复用（全局共用）
  └── 是UI元素？ → 复用（全局共用）
```

## 技术规范

### 角色形象（characters）

| 项目 | 规格 |
|------|------|
| 格式 | PNG（透明背景）|
| 分辨率 | L0-L4: 256x256px，L5-L9: 512x512px |
| 精灵蛋 | 256x256px |
| 色彩模式 | sRGB |
| 文件命名 | `{species}_{level}.png`（如 `fox_L3.png`）|
| 存储位置 | `characters/{系别}/{种类}/` |

**图层要求：**
- 角色主体居中，留白四周至少 10%
- 无背景，Alpha 通道完全透明
- 阴影可选（角色脚下柔和阴影）

### 动画资源（animations）

| 项目 | 规格 |
|------|------|
| 格式 | Lottie JSON（Bodymovin 导出）|
| 帧率 | 30fps |
| 循环 | idle 动画循环，upgrade/interact 不循环 |
| 文件命名 | `{species}_{level}_{type}.json` |
| 存储位置 | `characters/{系别}/{种类}/animations/{type}/` |

### 道具图标（items/icons）

| 项目 | 规格 |
|------|------|
| 格式 | PNG（透明背景）或 SVG |
| 分辨率 | 64x64px |
| 风格 | 扁平化 + 白色描边 |
| 存储位置 | `items/icons/` |

### 道具动画（items/animations）

| 项目 | 规格 |
|------|------|
| 格式 | Lottie JSON |
| 帧率 | 30fps |
| 复用 | 全局复用（所有精灵共用）|
| 存储位置 | `items/animations/{use,obtain}/` |
| 定制 | 如需精灵专属效果，assetKey 加精灵标识 |

### 特效（effects）

| 项目 | 规格 |
|------|------|
| 粒子动画 | Lottie JSON，透明背景，全局复用 |
| 光效叠加 | SVG/PNG 256x256px，Alpha 渐变 |
| 存储位置 | `effects/` |

### UI 元素

| 项目 | 规格 |
|------|------|
| 图标 | SVG 优先，PNG 备选（32x32px）|
| 徽章 | PNG 64x64px |
| 存储位置 | `ui/` |

### 商城专属（items/shop）

| 项目 | 规格 |
|------|------|
| 面板背景 | PNG 800x600px（半透明深色背景）|
| 道具专属形象 | PNG 128x128px |
| 文件命名 | `{item_code}-shop.png` |
| 存储位置 | `items/shop/` |

## 精灵成长体系

| 等级 | 阶段 | 说明 |
|------|------|------|
| L0 | 精灵蛋 | 初始引导教程阶段，引导用户熟悉软件布局和功 |
| L1 | 幼年期 | 破壳孵化，体型小巧，引导完成进入此阶段 |
| L2-L4 | 成长期→成熟期 | 体型渐增，细节逐步丰富 |
| L5-L7 | 完全体 | 体型放大到 512px，色彩和细节完整 |
| L8-L9 | 终极形态 | 最大形态，全部细节展现 |

## 占位资产说明

当前所有资产均为 SVG 占位文件，标记为「占位图」。管理员可通过 `/admin/art-assets` 页面注册真实资产并替换占位文件。

## 前端读取

精灵组件通过 tRPC `sprite.getAsset` / `sprite.getSpriteImage` 端点获取已发布资产的 URL。读取优先级：

1. **art_assets 表**（已发布 + 激活的资产，按版本取最新）
2. **sprite_images 表**（旧系统，兼容已有数据）
3. **占位 SVG**（`/assets/sprites/characters/{species}/{variant}/L{level}.svg`）
